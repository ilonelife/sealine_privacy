# Sealine 개인정보처리방침

Sealine 카약 투어 앱의 개인정보처리방침 웹사이트입니다.

## 🚀 Vercel 배포 방법

### 1. Vercel CLI 설치
```bash
npm install -g vercel
```

### 2. 프로젝트 디렉토리로 이동
```bash
cd privacy-policy
```

### 3. Vercel 로그인
```bash
vercel login
```

### 4. 배포
```bash
vercel --prod
```

## 📁 파일 구조

```
privacy-policy/
├── index.html          # 메인 개인정보처리방침 페이지
├── vercel.json         # Vercel 배포 설정
├── package.json        # 프로젝트 정보
└── README.md          # 이 파일
```

## 🔧 로컬 개발

```bash
# 의존성 설치
npm install

# 로컬 서버 실행
npm run dev
```

## 📝 주요 특징

- **반응형 디자인**: 모바일과 데스크톱 모두 최적화
- **SEO 최적화**: 메타 태그 및 구조화된 데이터
- **보안 헤더**: XSS, CSRF 등 보안 취약점 방지
- **접근성**: 스크린 리더 및 키보드 탐색 지원
- **빠른 로딩**: 최적화된 CSS와 최소한의 JavaScript
