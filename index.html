개인정보처리방침

Sealine(이하 "본 앱")은 개인정보보호법에 따라 이용자의 개인정보 보호 및 권익을 보호하고 개인정보와 관련한 이용자의 고충을 원활하게 처리할 수 있도록 다음과 같은 처리방침을 두고 있습니다.

1. 개인정보의 처리목적

본 앱은 다음의 목적을 위하여 개인정보를 처리합니다:

  • 카약 투어 서비스 제공: 투어 생성, 참여, 실시간 위치 공유
  • 안전 관리: 응급상황 대응, 투어 중 참가자 위치 확인
  • 기본적인 앱 기능: 사용자 식별, 투어 기록 저장

2. 개인정보의 처리 및 보유기간

개인정보는 다음 기간 동안 보유됩니다:

  • 사용자 계정 정보: 앱 삭제 또는 계정 삭제 시까지
  • 투어 기록: 투어 종료 후 30일 (안전 관리 목적)
  • 위치 정보: 투어 종료와 동시에 삭제

3. 개인정보의 처리 항목

수집하는 개인정보

  • 필수: 닉네임, 이메일 (Firebase 인증)
  • 위치정보: GPS 좌표 (투어 참여 시에만)
  • 자동수집: 기기 정보, 앱 버전 (오류 분석용)

4. 개인정보의 제3자 제공

개인정보를 외부에 제공하지 않습니다. 단, 다음의 경우는 예외입니다:

  • 법령에 의한 요구가 있는 경우
  • 응급상황 시 생명 보호를 위해 구조기관에 위치 정보를 제공하는 경우

5. 개인정보 처리의 위탁

앱 서비스 제공을 위해 다음 서비스를 이용합니다:

  • Google Firebase: 사용자 인증, 데이터 저장
  • Google Maps: 지도 표시, 위치 서비스

이들 서비스는 각각의 개인정보처리방침을 따릅니다.

6. 개인정보 권리 행사

사용자는 언제든지 다음 권리를 행사할 수 있습니다:

  • 개인정보 열람, 수정, 삭제 요구
  • 개인정보 처리 정지 요구
  • 계정 삭제 (앱 내에서 직접 가능)

권리 행사 방법: 앱 내 설정 메뉴 또는 개발자 이메일로 연락주세요.

7. 개인정보의 파기

보유기간이 경과하거나 처리목적이 달성되면 개인정보를 즉시 삭제합니다.

  • 자동 삭제: 투어 종료 시 위치 정보 즉시 삭제
  • 계정 삭제: 사용자가 앱을 삭제하면 모든 데이터 삭제
  • 수동 삭제: 사용자 요청 시 즉시 처리

8. 개인정보의 안전성 확보

개인정보 보호를 위해 다음과 같은 조치를 취합니다:

  • 접근 제한: Firebase 보안 규칙을 통한 데이터 접근 제한
  • 최소 수집: 서비스에 꼭 필요한 정보만 수집

9. 위치정보 수집·이용·제공 사실 통지

중요: 본 앱은 카약 투어의 안전성과 서비스 품질 향상을 위해 위치정보를 수집합니다.

위치정보 처리 내용

  • 수집 목적: 투어 참가자 간 실시간 위치 공유, 응급상황 대응
  • 수집 방법: GPS (투어 참여 시에만, 사전 동의 필요)
  • 보유 기간: 투어 종료와 동시에 삭제
  • 제공: 응급상황 시에만 구조기관에 제공

11. 계정 삭제 및 데이터 삭제 요청

🗑️ 계정 삭제 방법

사용자는 다음 방법으로 계정과 관련 데이터를 삭제할 수 있습니다:

1. 앱 내에서 직접 삭제 (권장)

  • Sealine 앱 실행
  • "프로필" 탭 이동
  • "회원 탈퇴" 버튼 클릭
  • 확인 후 즉시 계정 삭제

삭제되는 데이터

  • 계정 정보 (이메일, 이름 등)
  • 생성한 투어 기록
  • 참여한 투어 기록
  • 활동 통계 및 기록
  • 앱 사용 기록

⚠️ 주의사항: 계정 삭제는 되돌릴 수 없으며, 모든 데이터가 영구적으로 삭제됩니다.
